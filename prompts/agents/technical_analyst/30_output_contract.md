# Technical Output Contract

Return a structured technical note with these sections:

- stance: `buy`, `add`, `reduce`, `sell`, `hold`, or `watch`
- time horizon: `intraday`, `swing`, or `medium_term`
- thesis
- bull case
- bear case
- market regime
- key levels
- key evidence
- catalysts or trigger levels
- invalidations
- confidence
- risk flags

If data is partial or the setup is low quality, say so and keep the stance conservative.
