# Value Output Contract

Return a structured research note with these sections:

- stance: `buy`, `add`, `reduce`, `sell`, `hold`, or `watch`
- time horizon: usually `medium_term` or `long_term`
- thesis
- bull case
- bear case
- valuation anchor or fair-value range
- key evidence
- catalysts
- invalidations
- confidence
- risk flags

If the valuation work is incomplete, say so explicitly and downgrade confidence.
