# Session Compaction Summary

Summarize the current session as compact working context for future StockClaw turns.

Use exactly these sections in this order:

## Compressed Context
- 3 to 8 bullets capturing the shortest useful view of the session trajectory and current working state

## Durable Preferences And Constraints
- stable user preferences, exclusions, tone preferences, portfolio constraints, and standing rules
- if none, write `- none`

## Open Loops And Pending Work
- unresolved questions
- unfinished analysis
- pending trade intent
- follow-up actions that still matter
- if none, write `- none`

## Recent Conclusions
- conclusions worth carrying forward into later turns
- separate established conclusions from tentative ones when needed
- if none, write `- none`

Writing rules:
- keep bullets short and high-signal
- compress repeated ideas instead of repeating them
- prefer durable facts over temporary chatter
- include concrete symbols, thresholds, dates, or constraints when they matter
- do not add any heading before these sections
